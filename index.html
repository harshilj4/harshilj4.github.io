Project 01 · Sensors / Control / Automation
Automatic Plant Watering System

A fully functional sensor-to-actuator control loop using Arduino and MATLAB. The system reads soil moisture in real time, applies threshold logic, and switches a pump via MOSFET — no manual watering required.

  • Soil moisture sensor calibration and threshold decision logic
  • MOSFET-based pump driver circuit for safe high-current switching
  • MATLAB serial interface for live plotting and system state feedback
  • Emergency stop logic and state-machine control flow

Project 02 · Airframe / Electronics / Aviation
RC Airplane — Scratch Build

Designed and assembled a flyable twin-motor RC aircraft from a foam glider airframe. Every decision — motor placement, prop pitch, CG, battery routing, and wiring — had immediate physical consequences on whether it flew straight or didn't fly at all.

  • Foam airframe modified for twin brushed motors and dual ESC setup
  • GD005/GD006 flight controller board wiring and integration
  • Motor, propeller, battery and receiver placement for balance
  • Multiple test iterations — tuning throttle response and CG position

Project 03 · Sensors / IoT / UX Research — In Development
Noise Tracker — Smart Study Space Monitor

A sensor system that classifies ambient noise levels in real time and reports occupancy status for campus study spaces — libraries, quiet rooms, common areas. Born out of a real frustration: showing up to a packed library with no way to know before you leave.

  • Sound sensor readings smoothed and classified into quiet / moderate / busy / full states
  • Pilot testing conducted in York University study spaces, libraries, and common areas
  • User surveys and observational research shaped the threshold and classification logic
  • Currently building app + backend infrastructure for live status updates — check any space remotely before heading there
  • Exploring Wi-Fi signal density as a supplementary passive occupancy signal (no microphone required)
